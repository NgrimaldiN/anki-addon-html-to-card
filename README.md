# Paste LLM Cards

An Anki add-on that adds a `Paste LLM Cards` button to the Add Cards window. You paste a JSON bundle generated by an LLM, and the add-on creates one or many notes directly in the currently selected deck without going through Anki's import flow.

## What It Does

- Adds a new button in the Add Cards window
- Accepts raw JSON or a fenced ` ```json ` block
- Optionally creates or updates a custom note type with fields, templates, and CSS
- Adds one or many notes directly to the deck currently selected in Add Cards
- Validates the payload before import

## Bundle Format

```json
{
  "version": 1,
  "note_type": {
    "name": "Beautiful Basic",
    "fields": ["Front", "Back", "Extra"],
    "templates": [
      {
        "name": "Card 1",
        "qfmt": "<section class='hero'>{{Front}}</section>",
        "afmt": "{{FrontSide}}<hr id='answer'><section class='answer'>{{Back}}</section>{{#Extra}}<aside>{{Extra}}</aside>{{/Extra}}"
      }
    ],
    "css": ".card { color: #1f2933; }"
  },
  "notes": [
    {
      "fields": {
        "Front": "<h1>Question</h1>",
        "Back": "<p>Answer</p>",
        "Extra": "<small>Optional detail</small>"
      },
      "tags": ["llm", "generated"]
    }
  ]
}
```

If `note_type` is omitted, the add-on uses the note type currently selected in Add Cards.

## Local Development

```bash
python3 -m pytest
python3 -m py_compile __init__.py addon/*.py
```

## Install

1. Copy this add-on folder into your Anki `addons21` directory.
2. Restart Anki.
3. Open Add Cards and click `Paste LLM Cards`.
